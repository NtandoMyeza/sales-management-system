# Sales Management System

Beginner CRUD application

## Features
- Add customer
- View customers
- Update customer details
- Delete customer

## Technologies Used
- PHP
- MySQL
- HTML/CSS

## How to Run
1. Install AMPPS
2. Start Apache and MySQL
3. Import database.sql
4. Place the project folder in C:\Program Files\Ampps\www 
5. Open the browser and go to localhost/sales management system
